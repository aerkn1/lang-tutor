## Normalized Grammar Tree

### Cases

#### Definite Articles in the Dative
- Source form: `Dativ`
- Rule summary: Definite articles change to mark the indirect object or a dative-governed phrase.
- Core pattern: `der -> dem`, `das -> dem`, `die -> der`, plural `die -> den`
- Example: `Ich gehe zum Arzt.`
- CEFR: A1
- CEFR note: verified as A1 from Goethe A1 grammar overview and A1 lesson material focused on dative articles.

#### Definite Articles in the Accusative
- Source form: `Akkusativ`
- Rule summary: Definite articles change to mark the direct object, especially the masculine article.
- Core pattern: `der -> den`, `die -> die`, `das -> das`
- Example: `Ich sehe den Mann.`
- CEFR: A1
- CEFR note: verified as A1 from Goethe A1 grammar overview and A1 lesson material focused on accusative articles.

### Prepositions

#### `in` with Accusative for Motion into a Bounded Space
- Source form: `IN Kuralı`
- Rule summary: Use `in` with the accusative when movement goes into an enclosed or bounded space.
- Core pattern: `in + accusative` for direction into a place
- Example: `Ich gehe in den Park.`
- CEFR: A1

#### `zu` with Dative for Destinations and People
- Source form: `ZU Kuralı`
- Rule summary: Use `zu` with the dative for going to a person or to a destination point without emphasizing entry into the interior.
- Core pattern: `zu + dative`, often `zum` or `zur`
- Example: `Ich gehe zum Arzt.`
- CEFR: A1

#### Contrast: `in` vs `zu` for Destination Meaning
- Source form: `Şampiyonlar Ligi Karşılaştırması`
- Rule summary: `in` emphasizes entering and being inside a place, while `zu` emphasizes reaching a destination or meeting point.
- Core pattern: `in + accusative` for entry, `zu + dative` for destination
- Example: `Ich gehe in den Supermarkt.` vs `Ich gehe zum Supermarkt.`
- CEFR: A1

### Verb Forms

#### `sein` as the Perfekt Auxiliary for Movement and Change
- Source form: `Sein (olmak) fiili`
- Rule summary: In Perfekt, many movement and change-of-state verbs use `sein` as the helper verb instead of `haben`.
- Core pattern: present-tense `sein` + past participle, for example `ich bin`, `du bist`, `er ist`, `wir sind`, `ihr seid`, `sie sind`
- Example: `Ich bin gefahren.`
- CEFR: A1
- CEFR note: explicit `Perfekt mit sein` appears in A1 grammar curricula, though the broader Perfekt system is reinforced again beyond A1.
